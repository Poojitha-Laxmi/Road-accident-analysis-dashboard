This project presents an interactive Road Accident Dashboard built using Excel. It analyzes accident data to identify patterns in casualty severity, road types, vehicle involvement, and environmental conditions.


🎯 Objectives
Analyze accident severity distribution (Fatal, Serious, Slight)
Identify high-risk road types and locations
Compare current year vs previous year trends
Understand impact of weather and lighting conditions
Provide actionable insights for safety improvements


📌 Key Insights
Majority of casualties are Slight (~84%), indicating high-frequency low-severity incidents
Most accidents occur on Single Carriageways (~309K cases)
Urban areas show higher accident concentration compared to rural
Peak accident trends observed during specific months (e.g., Oct–Nov)
Dry conditions account for most accidents, likely due to higher traffic volume


🛠️ Tools & Techniques Used
Microsoft Excel
Pivot Tables
Pivot Charts
Data Cleaning & Transformation
KPI Cards & Dashboard Design
Slicers for interactivity


📈 Dashboard Features
KPI summary (Total, Fatal, Serious, Slight casualties)
Year-over-Year comparison
Casualties by vehicle type
Road type analysis
Weather condition analysis
Urban vs Rural distribution
